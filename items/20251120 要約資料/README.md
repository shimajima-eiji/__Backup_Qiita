## 関連性の高い実績例:1

- プロダクト創出研修（システム開発xマーケティングxAI利活用
  - GPTベースの社用AI活用の開発推進をベースに自社サービスからGAP分析を行い、サービスソリューション開発と広告戦略実行し、研修というより社内ハッカソン
  - 受講者が3桁名を１会場で指導し、講師も2桁名という手厚い対応環境で実施。別ベンダー研修とも調整し研修運営の全体最適を推進
    メイン講師登壇もしつつ教材作成を並行し、研修定例MTGをサブ講師一同に展開、成績不振の方へのフォローアップ施策を行いつつかなり大掛かりなカスタマイズを実施したため、従来パッケージから大幅なアセスメント調整も対応

## 関連性の高い実績例:2

- チームエンジニア研修（エンベデッド／C#, Web開発／Java,TypeScript系, Python機械学習）
  - 個人開発経験（AI含む）と開発未経験（AIはばらばら）のスキル差が年々幅が広がってきている中で、受講者のスキルセットに応じたグループ分けと、各グループに合わせた課題設計を実施
  - チーム開発演習では、ビジネス要求仕様が曖昧な状態で発注し、これまたシステム要件が概要程度のミニプロジェクト（テンプレート）を用意。ビジネス要求定義からデプロイまでを2〜3週間で実施・講師はマネージャーとして伴走
  - 毎年「できるかどうかわからない」「チームメンバーに迷惑をかけてしまった（特に経験者）」という声を「どうやったら出来るか」「自分にできる部分やメンバーの開発支援の立ち回り（特に経験者）」を体験し、クライアント目線とベンダー目線を同時に学び、視座の差の体得を目指す

【凡例：日中の稼働調整難度】
◎：先約との調整が容易
●：他業務での調整日のため事前相談必須
▲：他業務で弱い拘束時間があるため、日中の対応が難しい
×：他業務で強い拘束時間があるため、日中対応不可

【凡例：稼働開始時間帯の目安】
終日　：原則どの時間帯でも調整可能
日中　：9:00〜17:00を想定していますが、19:00まで可能な場合があります（応相談）
業後　：17:00以降より調整可能
午後以外：午前または夜間で調整可能
