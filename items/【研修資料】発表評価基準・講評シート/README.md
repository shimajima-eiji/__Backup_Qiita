## 講評のポイント

### 褒めるときのフレーズ例

- 「ターゲットが非常に具体的でした。〇〇というペルソナ設定が良かったです」
- 「構図の選択理由が明確でした。テキストで学んだ〇〇構図をうまく使っていますね」
- 「NG要素チェックがしっかりできていました。特に〇〇への配慮が良かったです」
- 「調査結果をうまく企画に落とし込めていました」

### 改善点を伝えるときのフレーズ例

- 「ターゲットをもう少し具体的にすると、構図の選択もしやすくなりますね」
- 「構図を選んだ理由をもう少し説明できると、説得力が増します」
- 「NG要素チェックリストを使って、事前に確認する習慣をつけましょう」
- 「調査結果をもう少し具体的に企画に反映させると良いですね」

### 避けるべき言い方

- ❌「ダメでしたね」→ ⭕「ここを改善するともっと良くなります」
- ❌「間違っています」→ ⭕「こういう視点もありますね」
- ❌「全然できていない」→ ⭕「ここは今後の課題ですね」
