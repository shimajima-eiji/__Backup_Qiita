
# SNS撮影企画ワークショップ 進行台本

## 概要

| 項目 | 内容 |
|------|------|
| 対象テキスト | 3.2.1〜（p.69〜113） |
| 時間 | 45分 × 6限（計270分） |
| 人数 | 30名（5〜6人 × 5グループ） |
| 成果物 | 撮影企画書（Canvaスライド） |
| ツール | Gemini Deep Research、Canva |

---

## 全体タイムテーブル

| 限 | 時間 | 内容 |
|----|------|------|
| 1限 | 45分 | オリエンテーション＋調査開始 |
| 2限 | 45分 | 調査＋グループ内共有 |
| 3限 | 45分 | 企画書作成① |
| 4限 | 45分 | 企画書作成② |
| 5限 | 45分 | スライド仕上げ＋発表準備 |
| 6限 | 45分 | 発表＋講評 |

---

## 1限（45分）：オリエンテーション＋調査開始

### 0:00〜0:10｜オリエンテーション

**講師アクション**
- ワークショップの目的を説明
- 成果物（撮影企画書）のイメージを共有
- タイムスケジュールを提示

**伝えるポイント**
> 今日は「撮影企画書」を作成します。実際に撮影はしませんが、プロの現場で使われる企画書を作ることで、座学で学んだ内容を実践的に理解しましょう。

**配布資料**
- 企画書テンプレート
- Deep Research調査ガイド
- 業界テーマ一覧

### 0:10〜0:15｜グループ分け＋業界テーマ発表

**講師アクション**
- 5グループに分ける（くじ引き or 事前決定）
- 各グループに業界テーマを発表

| グループ | 業界 |
|----------|------|
| A班 | カフェ |
| B班 | アパレル |
| C班 | コスメ |
| D班 | 旅行 |
| E班 | 飲食（レストラン） |

### 0:15〜0:45｜Deep Research開始

**講師アクション**
- Deep Researchの使い方をデモ（5分）
- 調査ガイドに沿って検索を開始させる
- 各グループを巡回し、困っている人をサポート

**伝えるポイント**
> グループ内で調査テーマを分担してください。1人で全部やる必要はありません。

**チェックポイント**
- [ ] 全グループがGeminiにアクセスできているか
- [ ] 検索プロンプトが適切か
- [ ] 分担ができているか

---

## 2限（45分）：調査＋グループ内共有

### 0:00〜0:25｜Deep Research継続

**講師アクション**
- 各グループを巡回
- 調査が進んでいないグループにヒントを出す
- 参考になる事例を見つけたグループを褒める

**よくある質問と対応**
| 質問 | 回答 |
|------|------|
| 検索結果が出ない | プロンプトを具体的にしてみて |
| どれを参考にすればいい？ | 自分たちの業界に近いものを選んで |
| URLが見つからない | レポート内のリンクをチェックして |

### 0:25〜0:45｜グループ内で調査結果を共有

**講師アクション**
- 「ここから20分で調査結果を共有してください」と指示
- 各グループが企画書のどこに何を反映するか決めるよう促す

**伝えるポイント**
> 調査した内容を、企画書テンプレートのどのセクションに書くか整理してください。

---

## 3限（45分）：企画書作成①

### 0:00〜0:05｜企画書の項目確認

**講師アクション**
- テンプレートの構成を再確認
- 優先的に埋めるべき項目を指示

**優先順位**
1. 基本情報（グループ名、業界）
2. 企画の目的（ターゲット、KPI）
3. 撮影対象
4. 構図・アングル

### 0:05〜0:45｜企画書作成

**講師アクション**
- 各グループを巡回
- 議論が停滞しているグループにヒントを出す
- テキストの該当ページを参照するよう促す

**声かけ例**
- 「ターゲットは具体的に決まった？」
- 「構図を選んだ理由は説明できる？」
- 「テキストのp.74、p.89を見てみて」

**チェックポイント**
- [ ] 全グループが作業を開始しているか
- [ ] 特定の人だけが作業していないか
- [ ] テキストを参照しているか

---

## 4限（45分）：企画書作成②

### 0:00〜0:30｜企画書仕上げ

**講師アクション**
- 残りの項目を埋めるよう促す
- NG要素チェックリストの確認を忘れないよう注意

**残りの項目**
- 加工・フィルター方針
- NG要素チェック
- 参考イメージURL
- キャプション案

**声かけ例**
- 「NG要素チェックは必ずやってね」
- 「参考URLは3つ以上入れて」
- 「フィルターを使う理由・使わない理由を書いて」

### 0:30〜0:45｜香盤表に着手（余裕があれば）

**講師アクション**
- 企画書が完成したグループには香盤表の作成を促す
- 時間がなければスキップ可

**伝えるポイント**
> 企画書が完成したグループは、実際の撮影を想定した香盤表（タイムスケジュール）も考えてみましょう。

---

## 5限（45分）：スライド仕上げ＋発表準備

### 0:00〜0:30｜Canvaでスライド作成

**講師アクション**
- Canvaの使い方がわからない人をサポート
- スライド枚数の目安を提示（5〜7枚程度）

**スライド構成例**
1. タイトル（業界名、グループ名）
2. ターゲット＋目的
3. 撮影対象＋アイデア
4. 構図・アングル（スケッチ含む）
5. 加工・フィルター方針
6. NG回避ポイント
7. キャプション＋ハッシュタグ案

### 0:30〜0:45｜発表リハーサル

**講師アクション**
- 発表担当者を決めるよう指示
- 発表時間（7分）を意識させる
- グループ内でリハーサルをさせる

**伝えるポイント**
> 発表は7分です。全員が話す必要はありませんが、質疑応答は誰でも答えられるようにしてください。

---

## 6限（45分）：発表＋講評

### 0:00〜0:35｜発表

**進行**
- 各グループ7分 × 5グループ = 35分
- タイムキーパーを設置
- 時間超過は1分前に合図

**発表順**
1. A班（カフェ）
2. B班（アパレル）
3. C班（コスメ）
4. D班（旅行）
5. E班（飲食）

**講師アクション**
- 発表中はメモを取る
- 良い点を中心にコメントを準備

### 0:35〜0:45｜講評＋まとめ

**講評の観点**
- ターゲット設定の明確さ
- 構図選択の意図
- NG要素への配慮
- 座学内容の反映度
- 調査結果の活用度

**まとめで伝えること**
> 今日作った企画書は、実際の仕事でも使える形式です。「撮影しなくても企画力は鍛えられる」ということを覚えておいてください。
