## 6. NG要素チェックリスト（テキストp.112 Column6参照）

以下に該当しないことを確認してください。

### 法的リスク

- [ ] 著作権侵害の恐れがある素材を使用していない
- [ ] 肖像権侵害の恐れがある人物が映っていない
- [ ] 商標権侵害の恐れがあるロゴ・ブランドが映っていない

### 炎上リスク

- [ ] 差別的・偏見を助長する表現がない
- [ ] 過度な誇張・誤解を招く表現がない
- [ ] センシティブなトピック（政治・宗教など）に触れていない

### ブランドイメージ

- [ ] 企業・ブランドのトーン＆マナーに合っている
- [ ] 競合他社を否定する内容になっていない
- [ ] 品質の低い画像・動画になっていない
