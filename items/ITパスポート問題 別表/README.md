# Q85

```
○ 整数型: binaryToInteger(文字列型: binaryStr)
　整数型: integerNum, digitNum, exponent, i
　integerNum = 0

　for (i を 1 から binaryStrの文字数 まで 1 ずつ増やす)
　　digitNum = binaryStrの末尾から i 番目の文字を整数型に変換した値
　　　// 例: 文字 "1" であれば整数値1に変換

　　exponent = i - 1
　　integerNum = integerNum + digitNum × (2 の exponent乗)
　endfor

　return integerNum.
```
