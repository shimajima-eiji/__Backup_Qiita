ヒント（AI使ってよいです）

- COMの手をランダムに決めるには？

むりやりマーケティング課題っぽくすると

```
Webページ上に3つのボタン（A案：割引クーポン、B案：期間限定セール、C案：レビュー強調）を作ります。  
どれかのボタンがクリックされたら、ユーザータイプをランダムに1つ決めます。  
  - （価格に敏感／今すぐ欲しい／他人の評価重視）
ユーザータイプと選んだバナーの組み合わせから、あらかじめ決めた表に従って「相性：◎／○／△」を判定します。  
判定結果を「ユーザータイプ」「選んだバナー」「相性」を含む1行のメッセージとして画面に表示します。  
クリックイベント取得や表示更新などの処理は、すべてJavaScriptとDOM操作で実装します。
```
