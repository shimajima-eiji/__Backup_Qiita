# 発端
いちいちCMDとかChocolateyGUIで毎日最新化をポチポチ作業していた状態から開放されます。
これでウッカリ更新を忘れて古いものを使い続けてた！という事がなくなって快適になります。
（特にVSCode）

# 結論
`cup all`をタスクスケジューラに組み込めば完成です。
`cup`も中身は`choco`なので`choco`のパスが通ってないと使えません。つまり、管理者権限が必要です。
