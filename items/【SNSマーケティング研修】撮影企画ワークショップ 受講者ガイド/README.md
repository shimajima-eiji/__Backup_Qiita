
# SNS撮影企画ワークショップ 受講者ガイド

## ワークショップ概要

| 項目 | 内容 |
|------|------|
| 時間 | 45分 × 6限 |
| グループ | 5〜6人 × 5グループ |
| 成果物 | 撮影企画書（Canvaスライド） |
| 使用ツール | Gemini Deep Research、Canva |

---

## 作業のポイント（評価観点）

発表時に以下の観点で評価されます。作業中から意識しておきましょう。

| 観点 | 良い例 | 改善が必要な例 |
|------|--------|---------------|
| **ターゲット設定** | 年齢・性別・興味関心・課題が具体的 | 「若い女性」など曖昧 |
| **構図・アングルの意図** | 選んだ理由を説明できる | なんとなく選んだ |
| **NG要素の回避** | チェックリストを全て確認済み | チェック漏れがある |
| **調査結果の活用** | 参考URLが3つ以上、企画に反映 | 調査したが活かせていない |
| **発表・プレゼン** | 論理的で時間内、スライドが見やすい | 時間超過、説明が曖昧 |

---

## 全体の流れ

```
1限：オリエン＋調査開始
    ↓
2限：調査＋グループ内共有
    ↓
3限：企画書作成①
    ↓
4限：企画書作成②
    ↓
5限：スライド作成＋発表準備
    ↓
6限：発表＋講評
```

# STEP 1：調査（1〜2限）

:::note info
**この時間の目標**
- グループのテーマを理解する
- Deep Researchで業界の成功事例を調査する
- 調査結果をグループで共有する

**時間配分**
- 1限：オリエン10分 → テーマ確認5分 → 調査開始30分
- 2限：調査継続25分 → グループ内共有20分
:::

## 1-1. グループでテーマを確認

あなたのグループに割り当てられた業界を確認してください。

| グループ | 業界 | 想定企業 | ミッション |
|----------|------|----------|------------|
| A班 | カフェ | 都内5店舗のカフェチェーン | 季節のフルーツパフェを魅力的に伝える |
| B班 | アパレル | 20〜30代女性向けブランド | 春物コートを「着たくなる」投稿を企画 |
| C班 | コスメ | ナチュラル系コスメブランド | 新色リップの魅力を伝え「試したい」と思わせる |
| D班 | 旅行 | 国内旅行会社 | 温泉旅館の魅力を伝え「行きたい」と思わせる |
| E班 | 飲食 | イタリアンレストラン | ディナーコースの魅力を伝え「予約したい」と思わせる |

## 1-2. Gemini Deep Researchで調査

### アクセス方法

1. https://gemini.google.com/ にアクセス
2. Googleアカウントでログイン
3. 画面上部から「Deep Research」を選択

### 調査の役割分担

グループ内で分担すると効率的です：

| 担当 | 調査テーマ | 使うプロンプト例 |
|------|-----------|-----------------|
| A | 構図・アングル事例 | 「[業界名] Instagram 人気投稿 構図」 |
| B | 加工・フィルター傾向 | 「Instagram 画像加工トレンド 企業アカウント」 |
| C | NG投稿・炎上事例 | 「企業SNS 炎上事例 教訓」 |
| D | 業界の成功事例 | 「[業界名] Instagram 成功事例」 |
| E | ハッシュタグ・キャプション | 「[業界名] Instagram ハッシュタグ 効果的」 |

### 業界別の検索プロンプト例

**カフェ業界**
```
カフェ業界のInstagram投稿で、エンゲージメント率が高い投稿の特徴を調査してください。
- 人気の構図パターン
- 撮影アングル
- 色調・フィルターの傾向
- 効果的なハッシュタグ
について、具体例を含めて教えてください。
```

**アパレル業界**
```
アパレルブランドのInstagram投稿で成功している事例を調査してください。
- 商品撮影の構図
- モデル起用 vs 物撮りの使い分け
- 季節感の演出方法
- ストーリー性のある投稿の作り方
について教えてください。
```

**コスメ業界**
```
コスメブランドのSNSマーケティング成功事例を調査してください。
- 商品撮影のテクニック（スウォッチ、使用感など）
- ビフォーアフター投稿の効果的な見せ方
- UGC（ユーザー生成コンテンツ）の活用方法
について教えてください。
```

**旅行業界**
```
旅行会社・観光業界のInstagram投稿で反応が良い投稿の特徴を調査してください。
- 風景写真の構図
- 「行きたくなる」投稿の共通点
- 季節ごとのコンテンツ戦略
について教えてください。
```

**飲食業界**
```
飲食店・レストランのInstagram運用で成功している事例を調査してください。
- 料理写真の撮り方（構図、ライティング）
- シズル感を出すテクニック
- 店舗の雰囲気を伝える投稿
について教えてください。
```

## 2-5. NG要素をチェック（テキストp.112 Column6参照）

以下に該当しないことを確認してください。

### 法的リスク

- [ ] 著作権侵害の恐れがある素材を使用していない
- [ ] 肖像権侵害の恐れがある人物が映っていない
- [ ] 商標権侵害の恐れがあるロゴ・ブランドが映っていない

### 炎上リスク

- [ ] 差別的・偏見を助長する表現がない
- [ ] 過度な誇張・誤解を招く表現がない
- [ ] センシティブなトピック（政治・宗教など）に触れていない

### ブランドイメージ

- [ ] 企業・ブランドのトーン＆マナーに合っている
- [ ] 競合他社を否定する内容になっていない

# STEP 4：発表（6限）

:::note info
**この時間の目標**

- 各グループ7分

- **ターゲットを明確に**説明する

# 困ったときは

| 困りごと | 対処法 |
|----------|--------|
| Geminiにアクセスできない | グループ内でアカウントを持っている人に依頼 |
| 検索結果が浅い | プロンプトをより具体的にする |
| 構図が決まらない | テキストp.69〜を参照 |
| Canvaの使い方がわからない | 講師に質問 or Googleスライドで代替 |
| 時間が足りない | 優先順位の高い項目から埋める |
