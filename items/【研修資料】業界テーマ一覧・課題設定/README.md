## 各業界の課題設定

### A班：カフェ業界

**設定**
- 企業：都内で5店舗を展開するカフェチェーン
- Instagramフォロワー：5,000人
- 課題：新メニュー「季節のフルーツパフェ」の認知拡大

**ミッション**
> 季節のフルーツパフェを1投稿で魅力的に伝える企画を立案せよ

**ヒント**
- シズル感をどう出すか
- 季節感をどう演出するか
- 「食べたい」と思わせる構図は？

---

### B班：アパレル業界

## 共通の評価観点

どの業界でも、以下の観点で企画を検討してください：

1. **ターゲット設定の明確さ**
   - 誰に届けたいか明確か

2. **構図・アングルの意図**
   - なぜその構図を選んだか説明できるか

3. **NG要素の回避**
   - 炎上リスク、法的リスクを考慮しているか

4. **座学内容の反映**
   - テキストで学んだポイントが活かされているか

5. **調査結果の活用**
   - Deep Researchの結果が企画に反映されているか
