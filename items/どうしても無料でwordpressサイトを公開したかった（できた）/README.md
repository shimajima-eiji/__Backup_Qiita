### デプロイする
各サイトにファイルをアップロードする事になるでしょうが、githubと連携すればCI(GithubActions)が使えるので、これを使いましょう。
wordpressからgithubにpushできれば解決できそうです。
「wordpress git push」あたりで検索すると色々情報が出てきます。
